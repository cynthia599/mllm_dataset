# 多模态大模型安全数据集整理

本仓库整理多模态大模型安全、越狱攻击和安全评测相关的公开数据集，
覆盖图文、音频、视频和全模态数据。

本仓库不重新分发原始大规模数据，仅提供：

* 数据集介绍
* 论文出处
* 官方下载地址
* 下载脚本
* 数据读取示例
* 数据集整理汇报
