\- 数据集名称：VLGuard

\- 模态：图像 + 文本

\- 任务：视觉语言模型安全微调、安全评测与安全对齐研究

\- 论文：Safety Fine-Tuning at (Almost) No Cost: A Baseline for Vision Large Language Models：https://arxiv.org/abs/2402.02207

\- 会议：ICML 2024

\- 官方数据：Hugging Face：https://huggingface.co/datasets/ys-zong/VLGuard

\- 官方代码：GitHub：https://github.com/ys-zong/VLGuard
